# 📱 Análisis de Comportamiento de Usuarios en una App Móvil

## 📌 Descripción  
Análisis de datos de usuarios de una aplicación móvil para comprender el funnel de conversión y optimizar la experiencia de usuario.

---

## ❓ Problema Identificado  
- ¿En qué etapas del embudo los usuarios abandonan la app?  
- ¿Qué eventos son más frecuentes y cómo se relacionan?  
- ¿Existen diferencias en la retención según cohortes?  

---

## 🛠️ Metodología y Herramientas  
- **Lenguaje:** Python (pandas, matplotlib, seaborn, plotly)  
- **Metodología:** Embudo de conversión, análisis de retención, cohortes de usuarios  
- **Datos:** Eventos en la app (pantallas visitadas, compras, tutoriales completados)  

---

## 📈 Resultados y Conclusiones  
- Se detectó alta deserción en la transición *Carrito → Pago*.  
- El tutorial inicial influye en la retención de usuarios.  
- Las cohortes recientes presentan mejor conversión que las antiguas.  

---
